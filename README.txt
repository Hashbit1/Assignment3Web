-Autocare Tracker-

This is a small website that lets you keep track of your maintance/mod list
for your vehicles. This uses Node.js, Express, MongoDB, and EJS tmplates. 

-What you can do-

Add services records, see the previous history, edit you records 
also delte it with a conformation mesage after deleting. 

-Features-

Add a new service entry with:
- Vehicle name
- Service type 
- Service date
- Cost
- Mileage
- Notes
- View all service records in a table 
- Edit an existing service record
- Delete a service record with a confirmation page
- Shared layout with a navbar, footer

-Technologies Used-

- Node.js and Express.js for the server and routing
- MongoDB for the database and data model
- EJS for server-side HTML templates
- Bootstrap 5 for layout and basic styling
- CSS in public/css/styles.css
